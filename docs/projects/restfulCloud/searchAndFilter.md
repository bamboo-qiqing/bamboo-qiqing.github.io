Restful Cloud 查找默认是搜索所有已经缓存过的项目，并且可以根据Request Type类型进行筛选。
插件也提供了SOA接口的兼容，对与消费接口的消费端和提供端也做了区分。

## Restful Api的搜索
::: tip
Restful Api使用的快捷键是alt+q,开发者可以根据url查询，此处以[Ruoyi Cloud 版本](https://gitee.com/y_project/RuoYi-Cloud)为例。
:::
