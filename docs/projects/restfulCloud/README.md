# Restful Cloud
[Restful Cloud](https://plugins.jetbrains.com/plugin/18885-restful-cloud)是restful api工具插件，使用sqlite缓存所有已打开项目的url，可以使打开一个项目的同时搜索到其他项目的url。
由于sqlite的缓存，可以使得开发者更便捷的对所有项目进行url的统计和soa接口的提供和消费的url检索。
