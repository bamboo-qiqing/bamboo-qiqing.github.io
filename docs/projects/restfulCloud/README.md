# Restful Cloud
[Restful Cloud](https://plugins.jetbrains.com/plugin/18885-restful-cloud)是restful api工具插件，对标Smart-doc

- [快速开始](QuickStart.md)
- [搜索和筛选](searchAndFilter.md)
- [注释](describe.md)
- [配置](setting.md)
- [数据缓存](database.md)
## 支持框架
::: tip
 框架的支持并非完全为接口路径的框架，还包括了在解析响应和请求参数时，参数是否必须和字段长度限制的框架支持。
:::
- [x] Spring
- [x] Spring Cloud Openfeign
- [x] Swagger
- [x] Hibernate
- [x] Jackson
- [x] Fastjson
- [x] Java

## 支持tag

::: tip
JavaDoc tag的支持来源于 [Smart-doc](https://gitee.com/smart-doc-team/smart-doc)，Restful Cloud 也以Smart-doc的tag解析为标准，以此来实现接口文档的零侵入性，详情查看[Smart-doc javadoc的使用](https://smart-doc-group.github.io/#/zh-cn/start/javadoc?id=javadoc%e4%bd%bf%e7%94%a8)。

与[Smart-doc](https://gitee.com/smart-doc-team/smart-doc)不同的是，Restful Cloud 并非支持所有的来自于[Smart-doc](https://gitee.com/smart-doc-team/smart-doc)的tag。
:::
